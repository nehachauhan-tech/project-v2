# Design System - AI Roleplay Chat

## Overview
A premium, modern, and minimal design system for an anonymous AI and random user chat application. Optimized for "WOW" factor and visual excellence.

## Visual Tokens
- **Background**: #eeeeee (Soft Light Gray)
- **Primary Accent**: #000000 (Pure Black)
- **Secondary Accent**: #ffffff (Pure White)
- **Typography**: 
  - Font Family: "Plus Jakarta Sans" (Google Font)
  - Headings: Bold, clean, modern.
  - Body: Regular, high readability.
- **Surface**: Glassmorphic cards with subtle shadows and rounding (16px).
- **Buttons**:
  - Primary: Black background, white text, 8px rounding, hover scale animation.
  - Secondary: Ghost style with black borders.

## Component Principles
- **Micro-animations**: Use Framer Motion for smooth transitions and hover effects.
- **Glassmorphism**: Use backdrop-blur for overlays and sidebars.
- **Spacing**: Generous whitespace for a premium "editorial" feel.
